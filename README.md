# jumpchess
制作的一个机械手臂控制的人机器中国跳棋博弈程序

有兴趣的可以添加钉钉号：simomo

设计目标：
1、通过摄像头识别物理棋盘跳棋棋子，并且映射到jumpchess的棋子内存中。
2、可以计算跳棋最佳方案。
3、自动指挥机械手臂按照最佳方案移动跳棋棋子。
4、能够判断输赢。
5、可以对人类的异常走棋进行反馈。
6、软件通过界面可以实时显示走棋的过程。

第一阶段目标：
1、实现一个纯软件的跳棋棋盘和可以在进行走棋的UI软件。----------------------------finished

第二阶段目标：
2、进行图像识别，可以正确的将家庭自然灯光下的棋盘使用摄像头识别正确，并能更新到软件的数据结构中。

第三阶段目标：
3、软件指挥机械手臂可以准确移动棋子

第四阶段目标：
4、可以正常完成人和机器的跳棋对弈

第五阶段目标：
5、升级下棋的对弈算法，让跳棋棋力增加。
